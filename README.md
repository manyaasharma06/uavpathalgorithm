# Impl1 C Program

## Description
This repository contains a C program named `impl1.c`.
It can be compiled and executed using a standard C compiler.

---

## Requirements
- GCC compiler (or any C compiler)
- Terminal (macOS/Linux) or Command Prompt (Windows)

---

## How to Run on macOS / Linux

### Step 1: Open Terminal and navigate to the folder
cd path/to/your/folder

### Step 2: Compile the program
gcc impl1.c -o impl1

### Step 3: Run the program
./impl1

---

## How to Run on Windows

### Step 1: Open Command Prompt and navigate to the folder
cd path\to\your\folder

### Step 2: Compile the program
gcc impl1.c -o impl1

### Step 3: Run the program
impl1.exe

---

## Notes
- Make sure `impl1.c` is in the current directory.
- Ensure GCC is installed and added to your system PATH.
